# Photo-booth app
This is a PyQt GUI application for an automated photo-booth.

## Software requirements
* Windows operating system
* Python 3.11
* PyQt5
* opencv-python
* pywin32

## Hardware requirements
* Nikon D70s digital camera
* USB webcam
* Touchscreen
* Printer

## Usage
`python.exe main.py`
